Activity 4.2.1. Solving Forced Second-Order Linear Differential Equations.

Find (1) a particular solution and (2) a general solution for each of the following differential equations.
(a)
\(x'' + 4x' - 21x = 2 e^{4t}\)
(b)
\(x'' + 4x' - 21x = 3e^{3t}\)
(c)
\(x'' - 4x' + 20x = 3 \sin 3t\)
(d)
\(x'' - 4x' + 20x = 2 \cos 4t\)
(e)
\(x'' - 14 x' + 49 x = \sin 3t\)
