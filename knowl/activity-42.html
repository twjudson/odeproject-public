Activity 6.1.2. Finding Inverse Transforms.

Find the inverse Laplace transform for each of the following functions.
(a)
\(F(s) = \dfrac{2}{s^2}\)
(b)
\(F(s) = \dfrac{1}{s^4}\)
(c)
\(F(s) = \dfrac{2}{(s - 3)^2 + 4}\)
(d)
\(F(s) = \dfrac{2s^2}{(s^2 + 1)(s - 1)}\)
(e)
\(F(s) = \dfrac{2s - 13}{s(s^2 - 4s + 13)}\)
