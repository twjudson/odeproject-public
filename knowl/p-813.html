Let
\begin{equation*} \frac{d {\mathbf x}}{dt} = {\mathbf f}(t, {\mathbf x}) \end{equation*}
be a system of differential equations such that \({\mathbf f}\) is continuously differentiable. If \({\mathbf x}_0\) is the initial value at time \(t_0\text{,}\) there exists a unique solution for the initial value problem on some open interval \(t_0 - \epsilon \lt t \lt t_0 + \epsilon\) for some \(\epsilon > 0\text{.}\)
