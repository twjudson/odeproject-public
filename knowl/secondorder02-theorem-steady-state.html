Theorem 4.2.3.

Suppose
\begin{equation*} x'' + px' + q x = g(t) \end{equation*}
has solution \(x = x_h + x_p\text{,}\) where \(p \gt 0\) and \(q \gt 0\text{.}\) If
\begin{equation*} x = x_h + x_p \end{equation*}
is the general solution to the equation, then
\begin{equation*} x = x_h + x_p \to x_p \end{equation*}
as \(t \to \infty\text{.}\)
